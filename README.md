# 🎮 Memory Card Game

## 📌 Description
The Memory Card Game is a fun and interactive web-based game where players flip cards to find matching pairs. The game tracks the time taken to complete all matches, enhancing the challenge and engagement.

## 🚀 Features
- 🃏 Card flipping with smooth 3D animations
- 🔀 Randomized card shuffling on each game start
- ⏱️ Real-time timer to track performance
- 🔄 Start and Reset game functionality
- 🎉 Result message upon completion
- 📱 Responsive and user-friendly design

## 🛠️ Technologies Used
- HTML5
- CSS3 (Flexbox, Transitions, 3D Transformations)
- JavaScript (DOM Manipulation, Event Handling)

## 🎯 How to Play
1. Click the **Start** button to begin the game.
2. Flip two cards at a time to find matching pairs.
3. If the cards match, they remain flipped; otherwise, they flip back.
4. Match all pairs to complete the game.
5. The total time taken will be displayed at the end.
